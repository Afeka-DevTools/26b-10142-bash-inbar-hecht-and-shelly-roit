## הוראות התקנה ושכפול הפרויקט
# 1. שכפול המאגר המקוון למחשב האישי
git clone [https://github.com/](https://github.com/)[username]/[repository-name].git

# 2. כניסה לתיקיית הפרויקט שנוצרה
cd [repository-name]
# 3. בשביל לבדוק אם יש לקבוץ הרשאות תריצו את הפקודה
ls -l script_name.sh
אם אין הרשאות תעשו את שלב 4
# 4. מתן הרשאות ריצה לכל קובצי הסקריפט
chmod +x *.sh
# 5. כדאי להריץ את הסקריפט תעשו את הפקודה הבאה
./script_name.sh
